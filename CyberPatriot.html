Windows CyberPatriot Competition To-Do List

  • *Note: This page is still under development, and only serves as a guide on what to do.

    • Forensics Questions:

  • Try to answer all of them correctly, as they award several easy points. Some online reaserch may be necissary.

    • User Account Management:

  • Disable the guest account
  • Remove unauthorized user accounts, but save the files
  • Demote any unauthorized administrators to standard users

    • Passwords:

  • Enforce that each user has a password
  • Enforce a secure minimum password length (8 characters)
  • Enforce that ach user's password meets the password complexity requirements
  • Ensure that a secure lockout threshhold exists

    • Audits:

  • Set audit policies to the given ones on packet. (Audit Credential Validation [Failure])

    • Login

  • Require [CTRL] + [ALT] + [DELETE] to be pressed before login (secure logon)

  • Security:

  • Enable firewall protection
  • Enable antivirus software
  • Uninstall hacking programs (such as Wireshark or Bit Comet)
  • Remove TX Backdoor (may not apply)
  • Remove prohibited games or music (.mp3) files
